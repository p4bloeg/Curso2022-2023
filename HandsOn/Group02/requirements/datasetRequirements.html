Dataset Requirements.


R1. Data are in one of the selected smart city domains.

The following datasets contain information focused on sustainable transportation system in the city of Madrid.
BiciMAD
Public Quick Charge Stations

R2. Data are available as a CSV file.

All datasets are available as CSV files, located at the csv folder.

R3. Data have an open license so they can be published.

Data sources have open license, this license can be checked by clicking the following link. Terms of use.

R4. Data can be easily linked with generic real-world entities (e.g., locations).

Datasets can be easily linked with real world as they are all based in Madrid city.

R5. Documentation exists for the data (optional).

There is documentation available via accesing the dataset links.

R6. Data can come from multiple sata sources (optional).

The information comes from different sources as EMT and "Dirección General de Planificación e Infraestructuras de Movilidad".
