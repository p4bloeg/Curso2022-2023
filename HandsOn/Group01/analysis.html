Analysis - Group 01

Data source

Dataset filename: puntos-de-recarga.csv The data sets we are using is regarding all the charge station for electric vehicle in the city of Alcobendas. The available information is: - Owner, address, longitudes, latitudes, number of connectors, type of connectors, price, cables and adaptors, bookable, schedule, timelimit.

Column Type Comments/Range Problems
Titular String Not unique
Situacion String Unique
Longitud Decimal -3.67 — -3.61
Latitud Decimal 40.51 — 40.55
Número de conectdores Integer 1 — 23 n/d?
Tipo de conectores String Not unique / Empty value 1 blank
Precio String Not unique n/d?
Cables y adaptadores String Not unique n/d?
Reservar String Not unique n/d?
Horario String Not unique n/d? / No standard for hours and days
Límite de tiempo String Not unique / Empty value A lots of blanks
