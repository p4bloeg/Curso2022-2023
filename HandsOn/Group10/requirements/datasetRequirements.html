Dataset Requirements

Dataset Requirements: "bases_bicimad.csv"

  1. R1. Data are in one of the selected smart city domains.

    Transportation.

  2. R2. Data are available as a CSV file

    Yes, bases_bicimad.csv

  3. R3. Data have an open license so they can be published.

    Yes. License aggrement: https://datos.madrid.es/egob/catalogo/aviso-legal

  4. R4. Data can be easily linked with generic real-world entities.

    The are fiels that are real-world entities such as: Distrito, Barrio, Calle, Longitud, Latitud, Direccion.

  5. R5. Documentation exists for the data (optional).

    No.

    More info: https://datos.madrid.es/sites/v/index.jsp?vgnextoid=eb8e993ae322b610VgnVCM1000001d4a900aRCRD

  6. R6. Data can come from multiple data sources (optional).

    No. The data come from "Dirección General de Patrimonio Cultural".
