App Description:

Our application give the user the tools to find fixed clean points in madrid, it also give extra information about this locations.

Requirements:

  • A search engine that allows the user to find and filter the fixed clean points around him or her.
  • The result of the search is going to be shown in an interactive map.
  • The user can touch or click in the different fixed clean points in the interactive map in order to have more information about this clean point.
