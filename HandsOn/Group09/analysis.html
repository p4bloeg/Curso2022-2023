Analysis

The Madrid Monuments dataset contains information on the name of the monuments information on the name of the different monuments, when they were built, when they were built, their author and where they are, both with street and number as well as through their coordinates

Data Analysis:

NAME: String It is the name of the monuments
DATE: String It is the date on which the monument was founded.
TYPE: String It the type of monument. EX: Statue
DESCRIPTION: String It contains some descriptions of the monument
AUTHORS: String It contains the name of the authors of the monument
CONTENT-URL: String It contains a url to the webpage that has information about the monument
STREET-NAME: String It contains the name of the street where the monument is located
CLASE-VIAL: String It contains the type of the street where the monument is located
NUM: String It contains the street number where the monument is located
NEIGHBORHOOD: String It contains the name of the neighborhood where the monument is located
PROVINCE: String It contains the name of the province where the monument is located
ZIP-CODE: String It contains the zip-code where the monument is located
DISTRICT: String It contains the name of the district where the monument is located
X-COORDINATE: INT It contains the x-coordinate where the monument is located
Y-COORDINATE: INT It contains the y-coordinate where the monument is located
